Kubeflow Pipelines UI Not Available

The kubeflow-port-forward sidecar is not reachable on port 8088.

docker compose -f docker/docker-compose.infra-platform.yml up -d kubeflow-port-forward nginx-proxy

If Kubeflow is not installed yet:

bash kubernetes/install-kubeflow-pipelines.sh

Direct URL: http://localhost:8088. Port 8081 is Jenkins.
